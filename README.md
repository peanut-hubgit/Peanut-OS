# PeanutOS v1.0.3

PeanutOS é um sistema operacional **inspirado no PyLinux**, recriado em Python com interface Tkinter para suportar **touch e click** em dispositivos móveis e desktops.

## Créditos

- Inspiração original: **PyLinux v1.0.3 (GPL v3)**  
- Maintainer & Tkinter Migration: **bangkkuser**

## Licença

PeanutOS é licenciado sob **GPL v3**.  
Isso significa que **qualquer versão distribuída deve ter o código-fonte completo disponível**, incluindo forks, experimentais ou estáveis.

## Permissões do Maintainer

- **bangkkuser** tem permissão de publicar **links das builds no Discord oficial do projeto**.  
- Builds lançadas devem ter **código-fonte completo disponibilizado**.

## Sobre o Projeto

PeanutOS foi criado como uma versão inspirada no PyLinux, mas com **interface própria em Tkinter**.  
O objetivo é fornecer uma experiência **simulada de Linux**, mantendo o espírito de aprendizado e experimentação.
