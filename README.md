# PeanutOS Classic v1.0

PeanutOS Classic é um sistema operacional **Em python**, feito para rodar direto no terminal de desktops.

## Créditos

- **@Peanut**

## Licença

PeanutOS é licenciado sob **GPL v3**.  
Isso significa que **qualquer versão distribuída deve ter o código-fonte completo disponível**, incluindo forks, experimentais ou estáveis.

## Permissões do Maintainer

- A pessoa que tiver interesse em ser maintainer tem permissão de publicar **links das builds no Discord oficial do projeto**.  
- Builds lançadas devem ter **código-fonte completo disponibilizado**.

## Links 
- Discord: [PeanutOS Discord oficial](https://discord.gg/wrSMwXHw)

- ## Comando para instalar python Windows  
- "winget install Python.Python.3"

- ## Comando para instalar python Mac OS
- **NECESSÁRIO HOMEBREW**
- "brew install python"

- ## Comando para instalar no Termux 
- "pkg update && pkg upgrade -y pkg install python -y"

- ## Requisitos 
- python 3.10+
