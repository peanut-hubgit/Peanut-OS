# Peanut OS Compact

O **Peanut OS Compact** é a edição leve do projeto, voltada a terminais, computadores modestos, ambientes educacionais e testes rápidos.

## Objetivos

- Consumir poucos recursos
- Inicializar rapidamente
- Funcionar com teclado em uma interface clara
- Servir como base simples para aprender Python
- Manter dependências opcionais ao mínimo

## Escopo inicial

- Tela inicial em terminal
- Menu principal
- Relógio e informações do sistema
- Calculadora
- Notas simples
- Configurações básicas
- Temas de texto
- Encerramento seguro da aplicação

## Fora do escopo inicial

- Kernel próprio
- Drivers próprios
- Substituição real do sistema operacional instalado
- Compatibilidade completa com aplicativos de Windows, Linux ou Android

## Requisitos planejados

- Python 3.10+
- `colorama`, `rich` e `pyfiglet` como dependências opcionais
