# Peanut OS Touch Interface

O **Peanut OS Touch Interface** é a edição experimental do Peanut OS para telas sensíveis ao toque, tablets, painéis e dispositivos com navegação por gestos.

## Objetivos

- Interface touch-first
- Elementos grandes e fáceis de tocar
- Navegação simples com poucos níveis
- Boa leitura em telas pequenas
- Adaptação entre orientação vertical e horizontal

## Recursos planejados

- Tela de bloqueio
- Tela inicial com atalhos
- Central de controle
- Gestos de deslizar
- Teclado virtual
- Configurações rápidas
- Aplicativos em tela cheia
- Modo quiosque opcional

## Regras de interface

- Áreas de toque amplas
- Sem controles minúsculos
- Feedback visual imediato
- Evitar excesso de janelas sobrepostas
- Compatibilidade com mouse durante o desenvolvimento

## Tecnologia

O framework gráfico ainda será escolhido após testes de desempenho, suporte a toque e compatibilidade com Android, Linux e Windows. A escolha deverá ser documentada antes do início da implementação principal.

## Estado
